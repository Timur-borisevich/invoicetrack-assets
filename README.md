# invoicetrack-assets

Публичные картинки для соцсетей InvoiceTrack (контент-календарь Instagram/FB).
Используются авто-постером как публичные image_url (Instagram Graph API).

- `assets/calendar/day-NN.jpg` — посты ленты 1:1 (1080×1080@2x)
- `assets/calendar/story-NN.jpg` — Stories 9:16 (1080×1920@2x)

Источник и генераторы — в основном маркетинг-репозитории (приватном).
